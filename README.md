## Setup Commands
 
### Setup
Show all setup command.

#### Usage
```
.setup
```

### Admin
Allow specific Roles/Users to execute specific Commands/All commands without ADMINISTRATOR roles.

#### Usage
```
.setup-admin   |
.setupadmin    |
.setup-mod     |  --> Follow the steps
.setupmod      |
.admin-setup   |
.adminsetup    |
```
